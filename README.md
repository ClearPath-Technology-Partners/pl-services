# PL Services LLC Website

Professional static website for PL Services LLC, a home and lawn service company based in Sandusky, Ohio.

## Features

- **Responsive Design** — Mobile-first, works on all devices
- **Service Pages** — Dedicated pages for lawn care, car detailing, junk removal, gutter cleaning, and snow removal
- **Photo Gallery** — Before/after gallery showcasing real jobs
- **Contact & Quotes** — Free quote form with CTA buttons
- **Google Maps** — Embedded map showing service areas
- **Reviews** — Customer testimonials and ratings
- **Component System** — Injected nav, footer, ticker, and quote form via `components.js`

## Pages

- `index.html` — Homepage
- `lawn-care.html` — Lawn care & landscaping service
- `car-detailing.html` — Mobile car detailing service
- `junk-removal.html` — Junk removal & moving service
- `gutter-cleaning.html` — Gutter cleaning & property services
- `snow-removal.html` — Snow removal service
- `about.html` — About PL Services
- `contact.html` — Contact form & FAQ
- `gallery.html` — Full before/after gallery
- `reviews.html` — Customer reviews

## Hosting

Hosted on Vercel. Push to `main` branch to auto-deploy.

## Technology

- Pure HTML5 / CSS3 / Vanilla JavaScript
- No build tool required — open `index.html` in browser or deploy as-is
- Google Fonts (Barlow Condensed, DM Sans)
- SVG icons (no emoji)
- Responsive CSS Grid / Flexbox

## Local Development

Open any `.html` file in a browser. No server or build step needed.

For live reload during development:
```bash
python -m http.server 8000
# Then visit http://localhost:8000
```

## Deployment to Vercel

1. Push to GitHub repository
2. Import repository in Vercel dashboard
3. No build configuration needed
4. Deploy on push to `main`
